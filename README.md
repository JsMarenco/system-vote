# System vote 

This is a simple voting system for learn more about next.js

## How to use

1. Clone this repo
2. Install dependencies
3. Run `npm run dev`
4. Open `http://localhost:3000`

## How to contribute

1. Fork this repo
2. Create a new branch
3. Make your changes
4. Create a pull request
